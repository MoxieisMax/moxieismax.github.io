# Affilaite — Jekyll site

This repository contains the original Jekyll site. A React port was created previously and archived to `archive/react-port`.

To run the Jekyll site locally (requires Ruby + Bundler + Jekyll):

```bash
cd C:\Users\Max\Desktop\Affilaite
bundle install
bundle exec jekyll serve --livereload
```

Or use Dockerizable commands if you prefer containerized builds.

If you need the React port, see `archive/react-port` or the `backup/react-port` branch on the remote repository.
